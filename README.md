# practice

> This is a sample Spring Boot application that uses several APIs

## 🚀 Features
* R2DBC: Uses Reactive Relational Database Connectivity (R2DBC) to connect to a PostgreSQL database.
* Multiple APIs: Exposes a multiple APIs:
    * REST
    * RSocket
    * GraphQL
    * gRPC
